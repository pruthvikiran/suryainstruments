TemCON 030617

Description

We, at Surya Instruments, manufacture temperature indicating cum controllers with multiple set points as per the requirement of customer for all types of sensor inputs and auxiliary potential supplies. As a single economical window for sourcing all types of controller, we can supply any type of sensor also along with required mechanical components and accessories for mounting temperature sensors. This is a standard size model with dual LED display with all configuration abilities to carry out sequentially faster. To produce highly reliable and standard accuracy of temperature indicators and controllers, we are well equipped and supported with expert as well as latest technology.

Also we offer all standard and custom built configurations to meet the needs of all customer’s temperature measurement applications.

Features :
  • - For Alarming Purpose at any set point / process parameter.
  • - For controlling Low / High levels of any process ex. Furnace Temperature etc.
  • - For HVAC or Air Conditioning systems.
  • - For Heat Exchangers Controlling.
  • - For regulating various control set points of Industrial Processes.
  • - For Laboratory Furnace Control Set points of all Processes and testings.
